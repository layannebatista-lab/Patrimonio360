# Patrimonio360
Gestão de ativos corporativos com Spring Boot e React. Solução Enterprise focada em conformidade LGPD, com auditoria imutável via Spring AOP, criptografia AES-256 e RBAC. Inclui automação logística com ViaCEP, termos de responsabilidade em PDF, specs dinâmicas com JSONB, BI analítico com Redis e testes automatizados com RestAssured e Cypress.
